WordPress Documentation plugin that parses markdown files and shows them in the admin and the customizer.

== Description ==

The help tab in the WordPress admin section is usually not a place users go to for help. This plugin adds a nice little help button on the bottom right corner of the WordPress admin and customizer.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the `huh_wp_docs_filter_doc_urls` filter hook to add help docs to any screen to configure the plugin
